# Heartbeat Checklist

- Calendar: Any events in the next 2 hours I should prepare for?
- Reminders: Any overdue or soon-due reminders?
- System: Is disk usage above 90%? Any critical process down?

If everything is fine, reply with exactly: HEARTBEAT_OK
